## Website Code Integration

### Potential New Page: "Find Your Mini Substation"

**URL:** `/find-your-substation/`

**Features:**
1. ZIP code lookup tool
2. Neighborhood dropdown
3. Show implementation timeline
4. Show anticipated response time

**Technical:**
- Simple HTML form
- PHP ZIP code validation
- JSON data file with 63 ZIP codes
- Conditional display based on implementation phase

**Sample Code Structure:**
```php
// Mini Substation Lookup
$zip_code = sanitize_text_field($_POST['zip']);
$substations = get_substation_data(); // JSON file

if (array_key_exists($zip_code, $substations)) {
    $station = $substations[$zip_code];
    echo "Your neighborhood: {$station['neighborhood']}";
    echo "Implementation: Year {$station['year']}";
    echo "Serving: {$station['zip_codes']}";
}
```
