# Stock Photo Search

Search Pexels for free stock photography (3M+ images).

**Usage:** `/stock-photo [search_query]`

## Instructions for Claude

1. Use MCP `pexels_search_photos` tool with the search query
2. Show top results with descriptions
3. Ask user which photo(s) to download
4. Use `pexels_download_photo` to download selected images
5. Report download location and file details

## Common Use Cases

- Find images for campaign website
- Get photos for social media posts
- Find Louisville-specific imagery
- Download professional headshots or event photos

**Example Keywords:**
- "Louisville neighborhood"
- "community meeting"
- "people voting"
- "public safety officers"
- "diverse community"
- "Kentucky landmarks"

## Notes

All Pexels photos are free for commercial use (no attribution required but appreciated).
